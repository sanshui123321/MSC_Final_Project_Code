This folder is used to run experiments in a normal network environment
The folder ids is used to store the serial number of the mobile phone
The Screenshot folder is used to store screenshots of successful experiments
The Furbo.txt file sets parameters for the experiment
Experiment.sh is the main script of the experiment, which calls compare_images.py code during operation
Experiment run results are recorded in furbo_result.txt
The pcap traffic file generated by the experiment is saved to the traffic_pcap folder
Pay attention to modifying information such as folder path and device parameters when running the script
